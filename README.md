# Gastei Bot 🤖

Bot do WhatsApp para lançar gastos no Gastei via mensagem.

## Variáveis de ambiente (Railway)

```
SUPABASE_URL=https://cbfqczriwzkrsoeudsyi.supabase.co
SUPABASE_KEY=sua_anon_key
ZAPI_TOKEN=seu_token_zapi
ZAPI_INSTANCE=sua_instancia_zapi
MEU_NUMERO=5512999999999
PORT=3000
```

## Comandos

- `Mercado 150` → lança saída
- `Uber 25,50` → lança saída
- `Relógio 10x de 4k` → lança parcelas
- `entrada salário 8000` → lança entrada
- `resumo` → resumo do mês
- `desfazer` → cancela último lançamento
- `ajuda` → lista comandos
